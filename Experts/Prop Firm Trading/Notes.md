# PUSH Simulation Files

All the current versions of the PUSH simulation files (To Be Updated in the future due to transfer to OOP)

## Current PUSH Versions

### CSV

    - Contains both Challenge and Funded simulations
    - Writes data from simulations to a CSV File
    - Has trinity functionalities

### Simulations

    - Contains both Challenge and Funded simulations
    - Has trinity functionalities

### Trinity

    - Doesn't contain any automated simulation
    - Manages push risk management depending on input Stage and TradeNumber
    - Has trinity functionalities
    - Currently being used live on prop firms
